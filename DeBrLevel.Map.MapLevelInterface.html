Interface - Leveled Map

Based on the overlay MapExt we defined interfaces for maps with shift and Wf. The shift function is applied on all leveled elements (keys and/or data), and a map is well-formed under k if all leveled elements in it are also well-formed under k. For each interface we specify via the notation ?K/?D the configuration of the map where K stands for keys and D for data. ? can be OT for ordered type, ET for equality type and Lvl for leved type.
